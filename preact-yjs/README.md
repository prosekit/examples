# preact-yjs

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/preact-yjs)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/preact-yjs)

Run the example locally with:

```bash
npx degit prosekit/examples/preact-yjs preact-yjs
cd preact-yjs
npm install
npm run dev
```
