# solid-gap-cursor

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/solid-gap-cursor)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/solid-gap-cursor)

Run the example locally with:

```bash
npx degit prosekit/examples/solid-gap-cursor solid-gap-cursor
cd solid-gap-cursor
npm install
npm run dev
```
