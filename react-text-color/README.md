# react-text-color

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/react-text-color)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/react-text-color)

Run the example locally with:

```bash
npx degit prosekit/examples/react-text-color react-text-color
cd react-text-color
npm install
npm run dev
```
