# vue-emoji-rules

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/vue-emoji-rules)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/vue-emoji-rules)

Run the example locally with:

```bash
npx degit prosekit/examples/vue-emoji-rules vue-emoji-rules
cd vue-emoji-rules
npm install
npm run dev
```
