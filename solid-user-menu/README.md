# solid-user-menu

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/solid-user-menu)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/solid-user-menu)

Run the example locally with:

```bash
npx degit prosekit/examples/solid-user-menu solid-user-menu
cd solid-user-menu
npm install
npm run dev
```
