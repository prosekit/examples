# solid-save-html

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/solid-save-html)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/solid-save-html)

Run the example locally with:

```bash
npx degit prosekit/examples/solid-save-html solid-save-html
cd solid-save-html
npm install
npm run dev
```
