# svelte-yjs

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/svelte-yjs)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/svelte-yjs)

Run the example locally with:

```bash
npx degit prosekit/examples/svelte-yjs svelte-yjs
cd svelte-yjs
npm install
npm run dev
```
