# svelte-text-color

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/svelte-text-color)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/svelte-text-color)

Run the example locally with:

```bash
npx degit prosekit/examples/svelte-text-color svelte-text-color
cd svelte-text-color
npm install
npm run dev
```
