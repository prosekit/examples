# preact-text-align

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/preact-text-align)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/preact-text-align)

Run the example locally with:

```bash
npx degit prosekit/examples/preact-text-align preact-text-align
cd preact-text-align
npm install
npm run dev
```
