# vue-list-custom-checkbox

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/vue-list-custom-checkbox)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/vue-list-custom-checkbox)

Run the example locally with:

```bash
npx degit prosekit/examples/vue-list-custom-checkbox vue-list-custom-checkbox
cd vue-list-custom-checkbox
npm install
npm run dev
```
