# preact-view-adapter

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/preact-view-adapter)

Run the example locally with:

```bash
npx degit prosekit/examples/preact-view-adapter preact-view-adapter
cd preact-view-adapter
npm install
npm run dev
```
