# vue-hard-break

A [ProseKit](https://prosekit.dev) example.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/vue-hard-break)
[![Open with CodeSandbox](https://assets.codesandbox.io/github/button-edit-lime.svg)](https://codesandbox.io/p/sandbox/github/prosekit/examples/tree/master/vue-hard-break)

Run the example locally with:

```bash
npx degit prosekit/examples/vue-hard-break vue-hard-break
cd vue-hard-break
npm install
npm run dev
```
