- Online demo: https://prosekit.dev/astrobook/stories/svelte/toolbar
- Run the example locally:

  ```bash
  npx degit prosekit/examples/svelte-toolbar svelte-toolbar
  cd svelte-toolbar
  npm install
  npm run dev
  ```
