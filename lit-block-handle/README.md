[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/lit-block-handle)

Run the example locally with:

```bash
npx degit prosekit/examples/lit-block-handle lit-block-handle
cd lit-block-handle
npm install
npm run dev
```
