[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/prosekit/examples/tree/master/preact-heading)

Run the example locally with:

```bash
npx degit prosekit/examples/preact-heading preact-heading
cd preact-heading
npm install
npm run dev
```
